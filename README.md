# Galeria de Fotos

Uma aplicação web React que exibe uma galeria de fotos com funcionalidades de busca.

## Funcionalidades

- Visualização de fotos em grid responsivo
- Busca em tempo real por nome das fotos
- Efeitos visuais ao passar o mouse sobre as fotos
- Layout responsivo para diferentes dispositivos

## Tecnologias Utilizadas

- React
- CSS3
- **Unsplash**: API de imagens gratuitas (https://unsplash.com)
